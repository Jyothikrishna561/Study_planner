The AI-Assisted Study Planner is a web-based productivity tool designed to help students organize their academic workload. By combining the power of Python (Flask), MongoDB, and OpenAI, this app doesn’t just list your tasks—it helps you understand which ones are most important.

Key Features:

Task Organizer: Create, view, and delete study tasks in a clean dashboard.

AI-Powered Prioritization: The app uses OpenAI to analyze your deadlines and task descriptions to suggest which subjects need your attention first.

Deadline Alerts: Visual cues tell you which tasks are overdue and which are coming up soon.

Productivity Insights: Get personalized tips generated by AI based on the types of tasks you are currently working on.

How to Use the App:

Add a Task: On the main dashboard, enter the name of your subject, the specific task (e.g., "Read Chapter 4"), and the due date. Click "Add Task."

View Your List: Your tasks will appear in a list sorted by their deadlines.

Get AI Advice: Click the "Prioritize" button (or the AI insight section) to let the app analyze your list. It will tell you which task is the highest priority based on the date and complexity.

Mark as Done: Once you finish a study session, you can delete or mark the task as complete to clear your dashboard.

Track Overdue Items: Keep an eye on the color-coded alerts that show you if you've missed a deadline.

Technical Requirements:

Python: The core language for the backend logic.

Flask: The web framework that runs the site.

MongoDB: The database that stores all your study tasks securely.

OpenAI API: The "brain" of the app that provides smart suggestions.

Privacy & Security:
This project is built with security in mind. All sensitive information, like your AI API keys, is stored in a private ".env" file that is never shared publicly on GitHub.
